# X Launch Thread Draft

## Post 1

我做了第一个公开建造的 Skill：

YIS Loop

重复不是循环，而是在同一方向上持续叠加。

它是给那些喜新厌旧、容易半途而废、做很多事但力量没有叠加的人用的。

GitHub：
https://github.com/yilee369/yis-loop

## Post 2

我以前也是这种人。

看到新东西就兴奋，开始很多方向，但每个方向都没有真正叠加起来。

后来我尝试聚焦半年，效果不是一点点变好，而是发生了质的提升。

健身、跑步、内容创作，一些半途而废的东西，也重新被我捡起来。

## Post 3

我后来意识到：

重复不是机械循环。

真正有效的重复，是每一次都在同一方向上施加一点力量。

像水滴石穿，像音乐 loop，回来时不是回到原点，而是带着上一轮的力量回来。

## Post 4

所以 YIS Loop 不做时间管理，也不讲鸡血自律。

它只问四个问题：

1. 你的极核是什么？
2. 什么动作值得重复？
3. 什么只是忙碌感？
4. 哪个 Loop 能让重复产生叠加？

## Post 5

它会把你的动作分成四类：

- Core action：直接叠加力量
- Support action：辅助但不能喧宾夺主
- Noise action：看起来有用但不叠加
- Escape action：用来逃避重复的动作

## Post 6

第一版支持三个场景：

1. 内容创作者：选题很多但没有主线
2. 健身跑步：半途而废后如何重启
3. 工作聚焦：每天很忙但没有沉淀成果

后续我会根据真实反馈继续改。
