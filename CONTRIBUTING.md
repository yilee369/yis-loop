# Contributing

YIS Loop improves through real feedback.

The most useful contribution is not praise. It is a concrete use case.

## Good feedback

Please include:

- Your original situation
- What YIS Loop diagnosed
- Which part felt accurate
- Which part felt wrong or vague
- What action you repeated for 7 days
- What signal changed after repetition

## Useful issue types

- The diagnosis was wrong
- The loop was too vague
- The do-not-do list was not strict enough
- A new scenario should be supported
- A real example can be added
