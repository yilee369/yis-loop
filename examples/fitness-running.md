# Example: 健身跑步半途而废后如何重启

## 输入

```text
/yis-loop

我以前健身和跑步都半途而废。
最近想重新开始，但我一想到完整计划就很烦。
我以前会买装备、研究训练计划、看视频，但真正能持续的很少。
我想知道这次怎么不再空转。
```

## 可能输出

```text
# YIS Loop Diagnosis

## Your Current Scattered State
你的问题不是不知道怎么健身，也不是缺训练计划。
你以前把“研究计划、买装备、看视频”当成了开始，但真正的核心动作没有被稳定重复。

## Dispersion Pattern
- Pattern: overlearning + tool hopping
- Evidence: 你会在开始前增加准备动作，让自己看起来已经进入状态，但身体没有被重复训练。

## Your Core For The Next 7 Days
For the next 7 days, repeat "完成一次 20 分钟低门槛训练或慢跑" to strengthen "重新建立身体和行动之间的信任", using "是否完成、体感、下一次阻力" after each loop.

## The 7-Day Loop
1. Repeat: 每天只做 20 分钟训练或慢跑，不追求强度。
2. Record: 写下是否完成、身体感受、最想放弃的时刻。
3. Adjust: 第二天只降低阻力，不增加复杂度。

## Do-Not-Do List
- 不买新装备。
- 不研究新训练体系。
- 不制定 90 天宏大计划。
- 不用体重变化判断这 7 天是否有效。

## One-Line Prescription
你要恢复的不是体能，而是“我会回来继续做”的证据。
```
