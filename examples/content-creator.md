# Example: 内容创作者选题很多但没有主线

## 输入

```text
/yis-loop

我做内容很散。
最近 7 天我发了 AI 工具、个人成长、读书笔记、商业案例、小红书运营。
每条我都觉得有价值，但账号没有形成主线。
我也想做公众号、小红书、视频号，还想做资料包和咨询。
我不知道该重复什么。
```

## 可能输出

```text
# YIS Loop Diagnosis

## Your Current Scattered State
你不是没有内容能力，而是每条内容都在服务不同身份。
AI 工具、个人成长、读书、商业案例、小红书运营都能讲，但它们没有共同指向一个清晰人群和结果。

## Dispersion Pattern
- Pattern: novelty drift + identity switching
- Evidence: 每个选题都能解释，但没有一个选题被重复到别人能记住你。

## Your Core For The Next 7 Days
For the next 7 days, repeat "写一个具体内容创作者遇到的分散问题" to strengthen "别人对你解决内容聚焦问题的认知", using "评论、收藏、私信、转发" after each loop.

## The 7-Day Loop
1. Repeat: 每天只写一个“内容创作者为什么做得很散”的具体场景。
2. Record: 记录哪类场景引发评论、收藏、私信。
3. Adjust: 第二天沿着信号最强的场景继续写。

## Do-Not-Do List
- 不新增平台。
- 不新增产品。
- 不写泛泛的 AI 工具推荐。
- 不写和内容聚焦无关的读书感想。

## One-Line Prescription
你不是选题少，你是没有把一个问题重复到别人记住。
```
