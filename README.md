# YIS Loop

> 重复不是循环，而是在同一方向上持续叠加。

YIS Loop 是一个帮助人从分散、喜新厌旧、半途而废中恢复聚焦的 AI Skill。

它不劝你自律，也不帮你做时间管理。

它只帮你做一件事：

```text
找到当前阶段值得重复的极核动作，
并把它设计成一个能持续叠加力量的 Loop。
```

## 为什么做这个 Skill

我以前也是很典型的喜新厌旧、无法聚焦、容易半途而废的人。

看到新东西就兴奋，开始很多方向，但每个方向都没有真正叠加起来。

后来我开始尝试聚焦，并持续做了半年。效果不是一点点变好，而是发生了质的提升。

一些曾经半途而废的事情，比如健身、跑步、内容创作，我也重新捡了起来，不再把它们当成一次性目标，而是把它们设计成可以不断重复、不断叠加的 Loop。

后来我看身边的同事和朋友，发现很多人不是没有能力，也不是不努力，而是没有找到自己的重复力量点。

他们做了很多事，但力量没有叠加。

所以我做了这个 Skill。

## 一句话

```text
没有极核的 Loop 是空转。
没有 Loop 的极核只是口号。
```

## 适合谁

- 总是喜新厌旧，开始很多事但很难坚持的人
- 内容、工作、学习、健身、生活都做得很散的人
- 不知道当前阶段到底该重复什么的人
- 一直换工具、换方法、换平台、换计划的人
- 想把一个方向做出叠加效应的人

## 不适合谁

- 只想要情绪鼓励的人
- 只想找更复杂计划表的人
- 不愿意暂停新想法的人
- 只想分析，不愿意做 7 天重复实验的人

## 30 秒试用

把下面这段发给支持 Skills 的 AI：

```text
/yis-loop

我现在做得很散。请帮我诊断：
1. 我现在的分散模式是什么
2. 哪个动作才是我当前阶段的极核
3. 哪些动作只是噪音或逃避
4. 未来 7 天我只应该重复什么 Loop

我的情况是：
{写下你最近 7 天做了什么，以及过去 6 个月半途而废过什么}
```

## 它会输出什么

YIS Loop 会给你一份诊断：

- 你现在的分散模式
- 你的极核候选
- 哪些动作是力量，哪些动作是噪音
- 未来 7 天只重复的 Loop
- 接下来 7 天不要做什么

## 核心方法

```text
收集动作 -> 识别分散模式 -> 找极核候选 -> 区分力量和噪音 -> 设计 7 天 Loop
```

## 安装

把 [`SKILL.md`](./SKILL.md) 放进你的 AI Skill 目录。

如果你使用 Codex，可以放到：

```text
~/.codex/skills/yis-loop/SKILL.md
```

- 使用后哪里有帮助
- 哪些诊断不够准
- 哪些场景不适配
- 哪些 Loop 真的产生了叠加

你可以通过 issue 提交反馈。
